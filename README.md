# 🚀 Invexa – Smart Inventory Management System

## 📌 Overview
Invexa is a real-time, AI-powered inventory management system designed to streamline procurement, sales, and operational workflows for small and medium-sized businesses (SMEs).

Traditional inventory systems rely on spreadsheets and disconnected tools, leading to inefficiencies, stock mismatches, and poor decision-making. Invexa solves this by providing a centralized, intelligent platform that ensures real-time visibility, automation, and accuracy across all operations.

---

## ✨ Key Features

### 📦 Inventory Management
- Add, update, and manage products in real time  
- Maintain accurate stock levels across workflows  
- Support structured and scalable inventory tracking  

---

### 🧾 Order Processing
- Create and manage sales orders  
- Track order lifecycle from creation to dispatch  
- Seamless integration with inventory  

---

### 🔄 Real-Time Stock Updates
- Automatic stock deduction upon order dispatch  
- Prevents stock inconsistencies  
- Ensures accurate inventory visibility  

---

### 📊 Dashboard & Insights
- Centralized dashboard with key performance indicators  
- Overview of inventory, orders, and operations  
- Enables data-driven decision-making  

---

### 🤖 Intelligence Layer (Conceptual)
- Analytics for performance insights  
- NLP-based command system  
- AI assistant for recommendations  

---

### 📑 Reporting & Monitoring
- Audit trails for activity tracking  
- Stock movement logs  
- Exportable reports and notifications  

---

## 🧠 Core Workflow Demonstration
1. Add products to inventory  
2. Create a sales order  
3. Dispatch the order  
4. Inventory updates automatically  

---

## 🏗️ System Architecture
Invexa follows a full-stack client–server architecture:
- Frontend handles user interaction and UI  
- Backend manages business logic and APIs  
- Database stores structured data  

---

## 🛠️ Technology Stack

### Frontend
- React.js  
- Tailwind CSS  

### Backend
- Node.js  
- Express.js  

### Database
- PostgreSQL  

### Security
- JWT Authentication  
- Role-Based Access Control  

---

## ⚙️ How to Run the Application

### 🔹 Prerequisites
- Node.js (v16+)  
- PostgreSQL  

---

### 🔹 Backend Setup
cd invexa-backend  
npm install  
npm run dev  

Runs on: http://localhost:5000  

---

### 🔹 Frontend Setup
cd inventory-dashboard  
npm install  
npm run dev  

Runs on: http://localhost:5173
